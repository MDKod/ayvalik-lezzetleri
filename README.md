 🍴 Ayvalık Lezzetleri

Ayvalık ve Cunda’nın sevilen lezzetlerini tanıtan modern bir web projesi.  
Kullanıcılar yemek kartlarına tıklayarak detayları modal şeklinde görüntüleyebilir. 📸✨
Proje sürecinde Google Labs Jules desteğinden yararlanıldı.

Projede:

- 🍤 Deniz ürünleri
- 🫒 Zeytinyağlılar
- 🧀 Ege mezeleri
- 🍨 Damla sakızlı tatlılar
- 📷 Instagram tarzı modal görüntüleme

özellikleri bulunmaktadır.


 🚀 Kullanılan Teknolojiler

- ⚛️ React 19
- ▲ Next.js 16
- 🎨 Tailwind CSS 4
- 💫 Modern App Router yapısı
